# IKEA 크롤링

이케아 웹사이트에서 제품 정보를 크롤링하는 프로젝트입니다.

## 기능

- 이케아 제품 목록 페이지에서 모든 제품 로드 (더보기 버튼 자동 클릭)
- 제품 타이틀 수집
- Selenium을 사용한 동적 웹 크롤링

## 설치

```bash
pip install -r requirements.txt
```

## 사용법

```bash
python ikea.py
```
